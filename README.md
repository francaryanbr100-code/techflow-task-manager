# TechFlow Task Manager

## Objetivo do Projeto

O TechFlow Task Manager é um sistema de gerenciamento de tarefas desenvolvido para uma startup de logística, com o objetivo de organizar fluxos de trabalho em tempo real, priorizar tarefas críticas e monitorar o desempenho da equipe.

## Escopo

O sistema permite:

* Cadastro de usuários
* Login no sistema
* Criação de tarefas
* Atualização de status das tarefas
* Exclusão de tarefas
* Visualização do progresso em quadro Kanban

## Metodologia Adotada

Foi utilizada a metodologia ágil Scrum com apoio do quadro Kanban para organização das tarefas.

## Tecnologias Utilizadas

* Python
* Flask
* GitHub
* GitHub Actions

## Controle de Qualidade

Foi implementado um pipeline de integração contínua com GitHub Actions para execução automática de testes.

## Gestão de Mudanças

Durante o desenvolvimento, surgiu a necessidade de adicionar prioridade nas tarefas (Alta, Média e Baixa), devido à necessidade do cliente de identificar atividades críticas com maior rapidez.

Essa alteração impactou o backlog e exigiu reorganização no quadro Kanban.
